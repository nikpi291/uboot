BILDSTIL ––

BILDSTIL

Bilder können als Freisteller in schwarz-weiß in das Layout eingebunden werden und frei auf dem Pattern platziert werden. Hierbei kann das Pattern mit dem Bild interagieren und das Pattern kann in Teilen über das Bild gelegt werden.
